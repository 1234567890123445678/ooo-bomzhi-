руководство по пользованию программой "ChtmlC"(computer html constructor)
При запуске можно ввести две команды:создать и редактировать.Команда создать создаёт по указанному пути файл и создаёт начало базовой структуры html документа.
Команда редактировать даёт куда больше возможностей,но сначало нужно указать путь к файлу.
список подкоманд команды редактировать:
img-позволяет разместить изображение в разрешении по умолчанию(необходимо указать источник).
img2-позволяет разместить изображение в разрешнии,которое ты укажешь(необходимо указать источник)(только одна сторона,из-за чего скорее всего изображение будет отображаться,не деформировавшись)
img3-позволяет разместить изображение в пользовательском разрешении(необходимо указать источник,сначало указывается ширина,а потом длина)
audio-позволяет разместить аудио на странице
video-позволяет разместить видео без режимов
video2-позволяет разместить видео со следующими режимами: a-автозапуск,l-повторное воспроизведени по окончанию звуковой дорожки,a,l-совмещонный режим.Есливам всего этого не надо-нажмите enter
gline-позволяет разместить горизонтальную линию
div-позволяет разместить текст
end-завешает построение вашей страницы
